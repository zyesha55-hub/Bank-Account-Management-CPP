# 🏦 Bank Account Management System (C++)

A robust C++ console application that demonstrates **Object-Oriented Programming (OOP)** principles. This project simulates basic banking operations with a focus on data security and validation.

## 🛠️ OOP Features Used
- **Classes & Objects:** Used to model a real-world bank account.
- **Encapsulation:** Account data (balance, name) is kept private to ensure security.
- **Constructors:** Automatic initialization of account details upon object creation.
- **Data Validation:** Logic to prevent negative deposits or over-drawing.

## 🚀 How to Use
1. Compile the `BankAccount.cpp` file using a C++ compiler (G++, MingW, etc.).
2. Run the executable to see the transaction flow in the console.
